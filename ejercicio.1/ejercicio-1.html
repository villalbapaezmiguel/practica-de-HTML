HTML

HTML son las siglas de Hyper Text Markup Language que se puede traducir como lenguaje de marca o marcado de inpertexto

El lenguaje HTML se emplea para crear paginas weps.Es muy facil ver el codigo HTML en una pagina wep, la opcion exacta cambia de un navegador a otro y tambien puede cambiar un aversion a otra de un mismo navegador , pero suelen tener un nombre similar

Historia de HTML

Los indicios del lenguaje HTML se remontan ala año 1990,cuando Tim Berners-Lee creo la primera pagina wep.

Versiones de HTML

Tim Berners-Lee definoo la primera version de HTML en el año 1991.

En la actualidad la ultima version de HTML, es HTML5.
